# Location Images

Place location-specific images here. Create subdirectories for each city:

- `eureka/` - Eureka city images
- `arcata/` - Arcata city images
- `fortuna/` - Fortuna city images
- `mckinleyville/` - McKinleyville images
- `trinidad/` - Trinidad images
- `ferndale/` - Ferndale images
- `rio-dell/` - Rio Dell images
- `blue-lake/` - Blue Lake images
- `hoopa/` - Hoopa images

Each subdirectory should contain:
- `hero.jpg` or `hero.webp` - Hero image for the location page (recommended: 1200x600)
- Optional: Local property examples or city landmarks

Images will be used in location cards and location page heroes with dark overlays for text readability.
