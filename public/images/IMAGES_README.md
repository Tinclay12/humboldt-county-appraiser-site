## Replacing with Your Own Images

When you're ready to replace these with your own AI-generated or professional photos:

1. Keep the same file names (e.g., `hero-background.jpg`, `hero.jpg`)
2. Recommended formats: WebP (best compression) or JPG
3. Recommended sizes:
   - Hero: 1920x1080 or larger (16:9 aspect ratio)
   - Services/Locations: 1200x600 (2:1 aspect ratio)
4. The Next.js Image component will automatically optimize them
