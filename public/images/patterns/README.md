# Pattern Images

Place geometric patterns, overlays, and decorative elements here.

Recommended formats:
- SVG for scalable geometric patterns
- PNG for overlay textures
- WebP for compressed background patterns

Common patterns:
- `geometric-overlay.svg` - Subtle geometric background pattern
- `mesh-gradient.png` - Gradient mesh overlay
- `texture.png` - Subtle texture overlay

These can be used as background elements to add visual interest without overwhelming content.
