# Hero Images

Place hero background images here. Recommended specifications:

- Format: JPG or WebP
- Dimensions: 1920x1080 (or larger, 16:9 aspect ratio recommended)
- File name: `hero-background.jpg` (or `.webp`)
- Content: Property exteriors, Humboldt County landscapes, or professional real estate imagery

The image will be used as a parallax background in the hero section with a dark overlay for text readability.
